NAME
====

Lithograph - Minimal static site generator.

SYNOPSIS
========

    % lithograph setup
    % lithograph write
    % lithograph build

DESCRIPTION
===========

Lithograph is a static site generator.
Lithograph supports markdown.
Lithograph supports GitHub Pages.
